3. Datahanteringspolicy

b. Datalagring. SkrivaPå lagrar Din avtalsinformation via Tjänsten kostnadsfritt i minst 2 år efter att avtalet undertecknats av alla parter. För avtal som inte undertecknats av alla parter lagras avtalet kostnadsfritt i 100 dagar efter att avtalet skapats. Efter 2 år för färdigställda avtal och 100 dagar för utkast, kan Du välja om Du vill betala den årliga lagringsavgiften, som SkrivaPå fritt kan justera från tid till annan, för att fortsätta lagra information för utvalda avtal. Användaren kan inom uppgivna tidsramar lagra upp till totalt 1GB avtalsinformation utan extra kostnad.

c. Dataåterhämtning. Det finns ett flertal funktioner i arkivet genom vilka Användaren kan hantera sina avtal, Användaren kan bland annat radera avtal från sitt konto. Information som inte längre finns tillgänglig via Användarens konto på grund av att (A) Användaren raderat informationen från sitt konto eller (B) Användaren sagt upp sig eller blivit uppsagd av SkrivaPå från sitt konto, kan mot en administrationsavgift som vi fritt kan justera från tid till annan, hämta ut uppgifter om sina avtal minst inom den tidsrymd som anges i punkt c.
